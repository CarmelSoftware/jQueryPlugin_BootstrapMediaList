jQuery Plugin Bootstrap Media Lists

This is an example of using the jQuery Plugin for Bootstrap Media Objects. Media objects (including images, videos, audios) are flexible and powerful components for displaying comment threads, articles lists, blog comments, Tweets, etc, that feature a left or right aligned image alongside textual content.
